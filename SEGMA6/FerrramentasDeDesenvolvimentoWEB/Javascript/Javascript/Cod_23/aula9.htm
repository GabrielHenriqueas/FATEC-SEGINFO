Aula - 9  Produção de Material Didático com JavaScript

Nesta aula estarei explicando como produzir um material didático com JavaScript. A idéia  é bem simples , você pode elaborar o conteúdo necessário e depois criar um mecanismo de avaliação.Esse mecanismo de avaliação funciona como uma página, onde o aluno responderá um questionário de múltipla escolha e ao término ele saberá exatamente quantas questões acertou e quantas errou.Dessa forma terá um botão onde ao clicar será enviado ao resultado para o e-mail do destinatário, neste caso o seu.

1º Passo: criando a página com o contexto a ser trabalhado

Inicie o Microsoft FrontPage 2000

Aqui você deve elaborar um arquivo comum com o assunto que deseja trabalhar com os alunos.No meu exemplo estou ilustrando do próprio JavaScript 

Após digitar todo o conteúdo, você deverá inserir um link , selecionando a palavra desejada, no meu caso será questões. Este link é feito com JavaScript, onde deverá abrir somente uma janela simples sem os controles.

Clique no Menu Ferramentas - Macro - Editor de scripts da Microsoft

Neste instante você terá acesso a  janela de código. Observe que o código a que me refiro é o HTML e justamente aqui é que vamos inserir o código em JavaScript.Para facilitar a digitação feche as janelas laterais, de forma que fique somente a janela de código com HTML conforme a ilustração abaixo:

Repare que a linha em destaque é o código a ser digitado.Localize a palavra Questões.Antes desta palavra você deverá digitar a linha abaixo:

<a href="Javascript:window.open('questões.htm','janela','status=yes,menubar=no,
                                                    toolbar=no,location=no,scrollbars=yes')">Questões</a>

Você se lembra deste código.... Bom aqui estou referenciando um arquivo de nome questões.htm que será aberto em uma nova janela do navegador, somente com a barra de status e rolagem. As demais foram desativadas por não terem necessidade.

Após esta etapa, salve a página com o nome de textojava.htm

Clique aqui e confira o modelo

2º Passo :Criando a página de questões

Esta é a parte mais trabalhosa, pois além de montar as questões, você terá que programar todos os controles. Vamos utilizar tudo que vimos no curso, funções com passagem de parâmetros, eventos, comando if..else entre outros.

Inicie uma nova página no FrontPage 2000. Para isso clique no botão Nova Página na barra de ferramenta e monte o formulário como no modelo abaixo:

.

A primeira parte, consiste em identificar os dados do aluno e logo em seguida a posição de acertos e erros.Agora vamos nomear os objetos pois é dessa forma que o JavaScript os identificará. Clique com o botão direito sobre a primeira caixa de texto e selecione propriedades do campo do formulário

Feito isto, formate o fundo da página em uma cor de sua preferência e depois escreva o título Questões. Formule a primeira questão e adicione dois botões de opção

Selecione o primeiro botão e selecione propriedades do campo do formulário.Altere os campos conforme a ilustração abaixo. Depois repita o mesmo procedimento para o botão falso

No campo nome do grupo digite opt1 e o valor inicial 0
No campo nome do grupo digite opt2 e o valor inicial 1

Formule a segunda questão e adicione 4 botões de ação

Agora altere a propriedade dos botões conforme a ilustração abaixo:

O Botão A terá o nome B1 e rótulo A  e assim teremos B2 para B, B3 para o C e B4 para D  

Por fim adicione o último botão só que este será do tipo submeter para enviar os dados.E salve esta página com o nome de questões.htm
