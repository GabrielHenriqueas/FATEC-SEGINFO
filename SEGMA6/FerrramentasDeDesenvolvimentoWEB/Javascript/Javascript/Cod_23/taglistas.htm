TAGS para criação de listas ordenadas e não ordenadas

Quando um navegador encontra a TAG <OL> que corresponde ao início de uma lista ordenada ou numerada, ele começa a mostrar cada item, que compõe a lista, precedido de números. Cada item passa a utilizar a TAG <LI>. A lista ordenada é finalizada com a TAG </OL>.  

Exemplos de listas ordenadas

<OL>
    <LI> Identificação
    <LI> Objetivo
    <LI> Relação de material utilizado
    <LI> Desenvolvimento Teórico
    <LI> Cálculos
    <LI> Conclusões
</OL>

O resultado está apresentado a seguir

  1. Identificação
  2. Objetivo
  3. Relação de material utilizado
  4. Desenvolvimento Teórico
  5. Cálculos
  6. Conclusões

Caso queiramos utilizar um tipo definido de marcador (1,i,I,a,A) devemos empregar o atributo TYPE juntamente com <OL> ou seja, < OL TYPE = i >. 

<OL Type =I>
        <LI> Movimento Uniforme
        <OL Type=a>
                <LI> Lei Horária
                 <LI > Representação Gráfica
                 <LI > Encontro de Móveis
</OL>

O resultado está apresentado a seguir

  1. Movimento Uniforme
    1. Lei Horária
    2. Representação Gráfica
    3. Encontro de Móveis

Podemos também, criar listas não ordenadas. Os símbolos empregados  são o  CIRCLE (circunferência), SQUARE (quadrado) ou DISC (círculo). Essas listas são iniciadas com a TAG <UL> e cada item emprega a TAG <LI Type="nome">, onde nome pode ser CIRCLE, DISC ou SQUARE.

Exemplos de listas não ordenadas

<UL>
    <LI> Português
    <LI> Matemática
    <LI> História
</UL>
Resultado:

Português
Matemática
História

<UL>
    <LI Type="square"> Português
    <LI Type="disc"> Matemática
    <LI Type="circle">História
</UL>
Resultado:

  • Português
  • Matemática
  • História
