Aula 3 - Comandos Básicos de Javascript

Vamos iniciar o estudo dos vários comandos de Javascript. Gostaria de lembrar que Javascript é sensível as letras maiúsculas e minúsculas e portanto você deverá digitar exatamente igual todos os comandos nos exemplos para que funcionem de forma correta.

Objeto document

O objeto document é um dos mais importantes no desenvolvimento de scripts em Javascript, pois atráves dele podemos enviar informações para o internauta.

document.write( ) : Este comando permite escrever qualquer informação na página que pode ser um texto ou o conteúdo de uma variável de memória.

Exemplo de impressão de um texto:

 document.write("Programando em Javascript");      Clique aqui para ver um exemplo

Exemplo de impressão do conteúdo de uma variável:

document.write("Olá "+nome+" Seja Bem Vindo !!!");    Clique aqui para ver um exemplo

document.bgColor : Este comando afeta diretamente a cor de fundo da página, ou seja, tem o mesmo efeito da tag em html <body bgcolor="nome_da_cor">

Exemplo

document.bgColor="red";     Clique aqui para ver um exemplo

document.fgColor : Este comando tem o efeito contrário ao document.bgColor, pois ele altera a cor da letra na página

Exemplo

document.fgColor="orange";    Clique aqui para ver um exemplo

document.title : Define um título para a página atual , ou seja, tem o mesmo efeito que a tag <title>..</title>

Exemplo

document.title="Curso de Javascript";  Clique aqui para ver um exemplo

document.lastModified : Exibe a data da última atualização da página

Exemplo

document.write(document.lastModified);    Clique aqui para ver um exemplo

Objeto window

O objeto window veja bem window e não windows é outro comando muito importante em Javascript pois através dele é que estabelecemos uma melhor interação com o internauta. É neste objeto que damos condições para a comunicação, como por exemplo, permitir a sua identificação no acesso a página.

window.alert( ) : Como já vimos anteriormente, este comando exibe uma caixa de diálogo para o internauta.Talvez você esteja perguntando:  Eu vi este comando escrito de outra forma ! .... Sem dúvida alguma você esta certo, o mesmo comando foi escrito assim  alert( ) que também é aceito na programação em Javascript.

Exemplo

window.alert("Curso de Javascript");

alert("Curso de Javascript");  Clique aqui para ver um exemplo

Neste exemplo você deve ter notado que a mensagem na caixa do alert( ) foi exibida em duas linhas. Isto foi possível graças ao código especial de quebra de linha "\n". Não se preocupe mais adiante no curso explicarei com mais detalhes.

window.prompt( ) : Este comando permite ao internauta realizar uma entrada de dados em uma caixa padrão e armazenar o dado digitado em uma variável de memória.Se você é um bom observador vai notar que já utilizei em exemplos anteriores, porém vou ilustrar um novo exemplo.

Exemplo

var nome=window.prompt("Por favor identifique-se....."," ");   Clique aqui para ver um exemplo

Assim como o alert ( ) este comando pode ser escrito no seguinte formato:

var nome=prompt("Por favor identifique-se...."," ");

window.open( ): Este comando é muito interessante, pois podemos abrir uma página em uma nova janela, além de controlar sua posição, tamanho e os controles que deverão aparecer na nova janela.Se você já acessou alguma página que abre automaticamente aquelas pequenas janelinhas com animações ou propagandas, já teve a oportunidade de ver como é a reação deste comando na página.Um bom exemplo disso é o portal do IG, se tiver uma conexão discada, vai perceber que estas janelinhas acabam atrasando a carga rápida da página no seu computador.

Exemplo

window.open("URL","nome_da_janela","Parâmetros");

URL : é o endereço da página a ser aberta

nome_da_janela : é um nome dado a nova janela a ser aberta que mais tarde poderá ser referenciada em código, usando o atributo target.

Parâmetros: aqui são mencionados quais são os controles de janela que deverão ser exibidos.A seguir explicarei cada um deles.

status (barra de status): possui dois valores yes(habilita a exibição) e no(desabilita a exibição)

location (barra de endereço) : possui dois valores yes(habilita a exibição) e no(desabilita a exibição)

toolbar (barra de ferramentas) : possui dois valores yes(habilita a exibição) e no(desabilita a exibição)

menubar (barra de menus) : possui dois valores yes(habilita a exibição) e no(desabilita a exibição)

scrollsbars (barra de rolagem) : possui dois valores yes(habilita a exibição) e no(desabilita a exibição)

left e screenX : definem a posição em pixels da janela a ser aberta a partir do lado esquerdo da janela do navegador. O left é utilizado no Internet Explorer e o screenX no Netscape.

top e screenY: definem a posição em pixels da janela a ser aberta a partir do aldo de cima da janela do navegador. O top é utilizado no Internet Explorer e o screenY no Netscape.

height (altura) : Controla a altura da nova janela. Este valor deverá ser informado em pixels.

width (largura) : Controla a largura da nova janela. Este valor deverá ser informado em pixels.
