TAGS para utilização de Áudio e Vídeo

Vídeo

Normalmente um arquivo de vídeo é incluído em um documento assim sendo, a TAG em navegadores Netscape  é: <EMBED SRC="clarear.avi"> enquanto que para o Internet Explorer é: <img border="0" dynsrc="clarear.avi">

Obs.: Esse atributo não apresentará o resultado esperado se o navegador não estiver configurado.

Áudio

Para navegador Netscape devemos usar a TAG: <EMBED SRC="sonata8.wav"> e para Internet Explorer usamos a TAG: <bgsound src="sonata8.wav" >
