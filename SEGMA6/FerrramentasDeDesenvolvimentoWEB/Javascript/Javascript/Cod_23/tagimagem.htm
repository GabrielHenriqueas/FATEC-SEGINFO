TAGS para inserção de imagens


Inserindo Imagem em um documento HTML
Para inserirmos uma imagem em um documento HTML devemos utilizar a TAG <IMG SRC>. Ela é única, ou seja, não necessita de TAG de finalização ( </img src> ).
O arquivo que contém a imagem a ser utilizada em sua Web page deverá estar armazenada, em seu provedor de acesso (yahoo, hpg, uol, vilabol, etc.) juntamente com o seu documento em HTML.
As imagens a serem incluídas devem estar no formato GIF ( Graphics Interchange Format ) ou JPEG ( Joint Photographers Expert Group ). O formato JPEG se enquadra melhor para imagens digitalizadas enquanto o formato GIF deve ser preferido para imagens com cores lisas, pois nesse caso a compressão seria melhor quando comparada à compressão feita pelo JPEG. 
 O formato JPEG é um padrão que pode comprimir imagens até 20% do seu tamanho original sem perda perceptível na qualidade. Em alguns casos as imagens no formato GIF apresentam melhor qualidade e/ou tamanho quando comparado com o formato JPEG.

Exemplo: <IMG SRC = "c:\hp\foto1.JPEG> => com esta TAG você irá inserir uma figura de nome foto1.jpeg que esta na pasta denominada hp armazenada no HD de seu computador.

Digitalizando uma imagem para ser utilizada na Internet

Quando formos escanear (digitalizar) uma imagem para utilizá-la na Internet devemos tomar o cuidado de escolher a resolução de 78 ppp (pontos por polegada), pois esta é a resolução dos monitores utilizados nos computadores (VGA e SVGA). Uma resolução maior implica numa imagem resultante de maior tamanho e isto acaba acarretando problemas de lentidão quando ela for aberta em sua página. Outro problema que precisa ser analisado é o número de cores empregado durante o processo de escaneamento(digitalização). Quanto maior o número de cores maior o tamanho da imagem obtida. O procedimento ideal é escanear com o número máximo de cores e depois, utilizando um software adequado ( Corel Photopaint, Adobe Photoshoh, etc.) reduzir o número de cores e em conseqüência o tamanho da imagem ficará menor.

Atributos empregados na TAG <IMG>

Atributo Descrição
Border Border = 4 - o número define a espessura da borda que contorna a imagem.
Alt Alt = " meu carro novo" - apresenta um texto alternativo quando o navegador é do tipo texto ou quando, nos navegadores gráficos, as imagens são desabilitadas.
Width e Height Width = "340" Eight = "260" - definem as dimensões da imagem. Esses atributos, na maioria dos editores de HTML já recebem valores automaticamente.  
Align Top - serve para alinhar um texto com o topo da imagem.
Bottom - serve para alinhar um texto com a parte de baixo da imagem. 
Right - serve para alinhar a imagem a direita de textos, imagens, etc.
Left - serve para alinhar a imagem a esquerda de textos, imagens, etc.
Middle - serve para alinhar a imagem com o meio da imagem.
Obs.: <BR CLEAR=RIGHT> libera o texto desses alinhamentos
Vspace e Hspace servem para definir o espaçamento vertical e horizontal, respectivamente, entre as imagens e os textos que circundam essas imagens.
Lowscr serve para sobrepor duas imagens de mesma dimensão(os atributos Width e Eight devem ser obrigatoriamente usados). O detalhe importante é que a primeira imagem a ser carregada deve ter um arquivo de tamanho bem menor que a segunda.

Algumas considerações:
a) É possível criarmos um link para outra página ou outro endereço na Internet a partir de uma imagem.
b) Uma imagem entrelaçada, no formato GIF, torna a visualização da página mais rápida, visto que, o navegador ao invés de trazer a imagem como um todo o faz por etapas, aumentando gradativamente a sua resolução. Para construímos uma imagem entrelaçada devemos nos valer de software como por exemplo o LView Pro (www.baixaki.com.br). O formato (gif87a) é o que apresenta melhor resolução para os arquivos  "Intelaced". 
c) Se desejarmos criar uma imagem com fundo transparente, o formato é gif89a. Existem vários softwares que permitem a criação deste tipo de imagem dentre os quais podemos destacar o LView Pro.
