Introdução ao JavaScript Básico

Javascript não é propriamente dito uma linguagem de programação , mas sim uma linguagem de script que funciona em página da Web na forma de recursos dinâmicos, ou seja, transformando uma página estática (HTML) em uma página dinâmica (HTML com JAVASCRIPT). Em outras palavras, podemos criar efeitos especiais nas páginas, controlar os dados digitados em um formulário, antes de enviá-lo ao seu destinatário, criar algumas animações, enfim dar um requinte especial nas páginas. Ela deve ser desenvolvida junto com HTML e que de início é um requisito básico para você desenvolver este curso.Além disso você deverá dominar a parte lógica, que considero essencial para poder compreender os exemplos que ilustrarei e explicarei durante o curso.

De início você deverá estar familiarizado com um editor de textos ( Bloco de Notas) ou qualquer editor de textos de sua preferência e um navegador Internet Explorer 5.0 ou superior. Caso utilize como navegador padrão o Netscape, poderá testar os scripts normalmente.Durante o curso estarei ilustrando exemplos com o Internet Explorer 5.5 SP2. Essa versão está disponível na seção de Donwload.

Configurações

Antes de iniciar o curso, é preciso verificar se o seu navegador está configurado para executar os scripts em Javascript.

1-) Clique no botão iniciar e logo em seguida aponte para a opção configurações

2-) Clique em Painel de Controle e logo em seguida dê um clique duplo sobre Opções da Internet

3-)Agora selecione a aba Avançado e localize o item Java VM. Observe que existem três itens os quais deverão estar todos selecionados.Caso tenha algum item sem seleção, selecione e clique no botão OK

4-) Reinicie o computador para que essas configurações tenham efeitos.Agora se todos os itens estavam selecionados, não há necessidade do reinicio do equipamento bastando clicar no botãoOK para fechar a janela propriedades da Internet .

5-) Volte na aba Geral e no item Página inicial , dê um clique no botão usar branco. Isto é só para facilitar a execução dos scripts no seu computador , principalmente seu você estiver conectado na Internet para não entrar outras páginas, pois isto levaria alguns segundos até a interrupção dessas páginas e carregar sua página que contém os scripts a serem testados.

 Você também poderá realizar as configurações dentro do próprio navegador Internet Explorer, basta clicar no menu Ferramentas e selecionar com um clique Opções da Internet após ter iniciado o navegador
