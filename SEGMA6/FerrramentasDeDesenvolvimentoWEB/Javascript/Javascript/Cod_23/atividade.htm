Atividade

Faça um material didático que tenha um conteúdo de seu domínio, o suficiente para ser ministrado em uma aula, com os recursos visto no curso.

Recomendações:

1-) Desenvolver uma página em HTML que tenha o conteúdo programático a ser ministrado em uma aula.

2-) Nesta página deverá ter um link para chamar uma 2ª página em HTML com objetos de formulário (botões de ação ou botões de radio) que represente as questões do conteúdo programático desenvolvido.

3-)Na página de questões deverá constar 10(dez) questões de múltipla escolha sobre o conteúdo programático. Nesta página deverá constar também os contadores de questões certas e erradas para o aluno visualizar.

4-)Esta página de questões não deverá ter nenhum controle de navegação exceto a barra de status e barra de rolagem.

5-)No final da página deverá ter um botão submeter para enviar os dados para seu e-mail.

6-)Ao término, salve em disquete  ou em CD-R e envie para a CETEC, via malote, muito bem embalado, aos cuidados do Prof. Oswaldo

Atenção: Caso tenha dúvidas na realização desta atividade entre em contado (ogiorgi@hotmail.com) 

OBS.: O não cumprimento desta fase implicará na não emissão do respectivo certificado.
