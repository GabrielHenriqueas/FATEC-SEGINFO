TAGS de formatação de Textos e caracteres


Formatação de Textos

A linguagem HTML permite alterar a cor o tamanho e o tipo de fonte utilizado no texto através da TAG <FONT>. 

Cor - para mudar a cor de um texto devemos utilizar a seguinte TAG:
<font color="red"> Esta frase será escrita na cor vermelha </font>
Observe que o nome da cor aparece entre aspas. Em lugar do nome podemos colocar o código hexadecimal da respectiva cor, ou seja, a cor vermelha (red) em hexadecimal fica "#FF0000".  

Existem tabelas que mostram a correspondência existente entre o nome da cor e a sua respectiva codificação hexadecimal. Editores mais modernos, de HTML, resolvem este problema com um simples clique em um ícone.


Tamanho - para mudar o tamanho de um texto devemos utilizar a seguinte TAG:
<font size=5> Tamanho da letra é 5 </font>
O tamanho normal da letra é 3, porém este tamanho pode variar de 1 a 8. Abaixo temos todos os exemplos. 

tamanho 1
tamanho 2
tamanho 3
tamanho 4
tamanho 5
tamanho 6
tamanho 7
tamanho 8

A TAG <font face> permite que você escolha o tipo de letra a ser utilizada em sua Home Page.
Exemplo: <font face=Arial> Este texto será escrito na fonte Arial </font>
Editores modernos, de HTML, permitem que você selecione o tipo de letra e o seu tamanho de uma maneira bem simples.

Formatação Física de caracteres (especifica o estilo que desejamos para o texto:negrito, itálico, etc.)

<B>... </B> Tudo o que estiver entre as duas TAGS é apresentado no estilo negrito.

<I> ... </I>   Tudo o que estiver entre as duas TAGS é apresentado no estilo itálico.

<U> ... </U> Tudo o que estiver entre as duas TAGS é apresentado no estilo sublinhado.

<SUP> ... </SUP> Tudo o que estiver entre as duas TAGS é apresentado no estilo sobrescrito

<SUB> ... </SUB> Tudo o que estiver entre as duas TAGS é apresentado no estilo subscrito

<STRIKE> ... </STRIKE> Tudo o que estiver entre as duas TAGS é apresentado no estilo riscado

<SMALL> ... </SMALL> Tudo o que estiver entre as duas TAGS é apresentado num tamanho menor de letra

<BIG> ... </BIG> Tudo o que estiver entre as duas TAGS é apresentado num tamanho maior de letra

Formatação Lógica de caracteres (segue o significado lógico de um texto como por exemplo um endereço de um e-mail)

<ADDRESS> ... </ADDRESS>  Utilizado para formatar endereços de e-mail, fazer referências. Normalmente o resultado é apresentado em itálico.

<CITE> ... </CITE> Utilizado para citações e títulos de um modo geral.

<CODE> ...</CODE> Utilizado para indicar trechos de códigos de programação.

<DFN> ... </DFN> Utilizado para descrever a definição de uma palavra, em geral o resultado é apresentado em itálico.

<EM> ... </EM>  Enfatiza o texto e o resultado é apresentado em itálico.

<KBD> ...</KBD> Indica uma entrada de informação via teclado.

<SAMP> ... </SAMP> Indica uma seqüência de caracteres como por exemplo um resultado.

<STRONG> ... </STRONG> Enfatiza o texto de maneira acentuada. O resultado é normalmente apresentado em negrito.

<VAR> ... </VAR> Indica o que o usuário deverá escrever. O resultado normalmente é mostrado em itálico.

Formatação de Blocos de Texto

<PRE> ... </PRE> Tudo o que estiver entre as duas TAGS é apresentado da mesma maneira que foi digitado, ou seja, mantendo-se as tabulações e as quebras de linha.

<BLOCKQUOTE> ... </BLOCKQUOTE> Esta TAG é empregada quando fazemos uso de longas citações.

<P> ... </P> Esta TAG indica a presença de um novo parágrafo e dá um espaçamento entre as linhas. Exemplo:<p> Este texto deve<p> ficar assim!

O resultado está apresentado abaixo:

Este texto deve

ficar assim!

<BR> ... </BR> Esta TAG proporciona a quebra de linha sem dar espaçamento entre elas. Exemplo:<br> As inscrições deverão ser feitas por <br> e-mail.

O resultado está apresentado abaixo:

As inscrições deverão ser feitas por
e-mail

Alinhamento de Texto

O texto pode ser alinhado pela esquerda, pela direita ou centralizado. O atributo empregado para o alinhamento é ALIGN que necessita vir acompanhado de um comando principal que pode ser <P>, <BR>, <Hn> e <DIV>.


<P ALIGN="Center"> ... </P>  Estas TAGs proporcionam um alinhamento centralizado do texto.
<P ALIGN="LEFT">   ... </P>  Estas TAGs proporcionam um alinhamento pela esquerda.
<P ALIGN="RIGHT">... <P>   Estas TAGs proporcionam um alinhamento pela direita.

Linhas de separação em um Texto

As linhas de separação são obtidas através da TAG <HR> seguida dos atributos: Size, Width, Align e noshade. Obs.: A tag <HR> não necessita da tag de finalização </HR>.

Exemplos:

<HR size=2 width=200 align=left> Insere uma linha horizontal com altura=2, comprimento=200 e alinhamento pela esquerda.

<HR Widht=40%> Insere uma linha horizontal que ocupa 40% do espaço disponível na tela.

 

<HR Width=10% Align=right noshade> Insere uma linha horizontal que ocupa 10% do espaço disponível na tela, alinhada pela direita e sem efeito tridimensional.

Mudando as cores de texto e fundo

Os navegadores disponibilizam até 256 cores para serem usadas como cor de fundo (background) ou de texto. Cada cor é composta por um código RGB(red,green,blue) que por sua vez é constituído de três números sendo que o primeiro representa a intensidade do vermelho (red), o segundo a intensidade do verde(green) e o terceiro a intensidade do azul(blue). Cada uma das três cores pode ter um número que varia de 0 a 255, porém ao invés de se trabalhar com um número na base decimal ele é convertido para a base hexadecimal, pois assim o computador trabalha melhor internamente.

Atuando na TAG <BODY> podemos definir as cores para: textos, links, fundo de página e papel de parede ( no formato marca d'água ou não). 

Atributos Descrição
TEXT define a cor dos textos ( o padrão é preto )
LINK define a cor dos links ( o padrão é azul )
ALINK define a cor do link quando clicado ( o padrão é vermelho )
VLINK define a cor dos links após terem sido visitado ( o padrão é roxo) 
BACKGROUND define a imagem a ser aplicada como papel de parede da página.
BGCOLOR define a cor de fundo ( o padrão é branco ). Obs.: Navegadores mais recentes aceitam nomes de cores, a saber: Olive, Black, White, Yellow, Red, Gray, Silver, Maroon, Purple, Blue, Fuchsia, Lime, Acqua, Teal. Se as cores não forem definidas o que vale é o padrão de cores do navegador.

Exemplos:
<body bgcolor="0,0,0"> => coloca a cor preta como fundo de sua página
<body text="255,255,255"> => coloca o texto em cor branca
<body link="blue" alink="red" vlink="yellow"> coloca o link na cor azul, o link acionado em vermelho e o link visitado em amarelo.
<body background = "c:/teste/imagem1.jpg"> coloca como papel de parede a figura de nome imagem1.jpg que está na pasta teste no HD.
