O que é JavaScript ?

Podemos definir JavaScript como uma linguagem de scripts que usada em parceria com HTML, permite que as páginas escrita para a Web se tornem dinâmicas e possibilitem uma interatividade com o internauta.É importante que você saiba que JavaScript não é a única linguagem de scripts para programar páginas da Web, existem outras como o VBScript, JScript da Microsoft.

Tanto JavaScript como o VBScript são linguagens para serem executadas no navegador do cliente, neste caso o navegador que está instalado em seu computador e que permite a você o acesso a internet. O VBScript é uma linguagem que foi desenvolvida para ser interpretada pelo Internet Explorer ou superior, deixando de lado os usuários do  Netscape.

Já o JavaScript foi desenvolvido pela Netscape Communications Corporation( o criador do próprio navagador Netscape que funciona perfeitamente bem no Internet Explorer.
