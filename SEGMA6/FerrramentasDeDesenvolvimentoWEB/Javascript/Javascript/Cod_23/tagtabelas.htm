TAGS para criação de Tabelas

As Tabela são muito utilizadas em páginas pessoais na Internet, pois permitem trabalhar as informações de uma maneira mais organizada. Toda a estrutura bem como os conteúdos de uma tabela devem ficar dentro das TAGs <TABLE> ... </TABLE>.
Atributos Descrição
<Caption> Define o título de uma tabela. <captiom> ... </caption>
<TH> Usado normalmente para especificar as células de cabeçalho da tabela. <TH>...</TH>
<TD> Usado para especificar as células de dados em uma tabela. <TD> ... </TD>
<TR> Usado para indicar o fim de uma linha na tabela. <TR> ... </TR>
Border Define a espessura da linha de borda da tabela <TABLE BORDER=6>
Align Controla o alinhamento do texto em relação às bordas laterais de uma célula. Pode ser utilizado com TH, TD ou TR. Ao ser aplicado junto com TR ele define o alinhamento de uma linha da tabela. Align aceita os atributos LEFT, CENTER ou RIGHT. <td align="center">
Valign Controla o alinhamento do texto em relação às bordas inferior e superior. Pode ser utilizado com TD e TH e aceita os atributos TOP, MIDDLE e BOTTOM. <TD VALIGN="TOP">...</TD>
Width Usado para definir o espaço que a tabela irá ocupar na tela.
Cellpadding Controla o espaçamento entre o texto e as bordas da célula.
Cellspacing Controla o espaçamento entre as células.
Bgcolor Controla a cor de fundo
Bordercolor Controla a cor da borda
Background Controla a imagem de fundo
Rowspan Permite mesclar linhas de uma tabela
Colspan Permite mesclar colunas de uma tabela

Exemplos de tabelas

<table border ="2" bordercolor="blue" width="100%">  =>Borda: espessura = 2, cor azul. Ocupa 100% da área disponível 
    <Caption> Relação de Professores/Escola </Caption>
                    <td align="center" bgcolor="red"> Nomes/Escolas</td> => Centraliza/cor vermelha de fundo
                    <th> Carlos Augusto</th>
                    <th> Ricardo Bueno </th>
                    <th> Regina Helena </th>
<tr> => Muda de linha
                    <th> Escola </td>
                    <td bgcolor="green"> ETE Fernando Prestes     </th>  => Coloca a cor verde no fundo
                    <td> <font color="#800080">ETE Lauro Gomes </th> => Coloca cor roxa na fonte         
                    <td > ETE Polivalente</th> 
<tr>
                    <th> Número de Matrícula </td>
                    <td align="center"> 12345</th> => Centraliza o número na célula
                    <td align="center"> 04002</th>
                    <td> align="center"21345</th>
</table>

Obs.: o que está destacado em amarelo é comentário
