Histórico

Internet: é um sistema de comunicação descentralizado e globalizado, constituído por redes de computadores interligados, ou melhor uma rede de rede, através da qual, qualquer pessoa que esteja conectada, tem acesso a um enorme conjunto de informações sem falarmos nos recursos e serviços que são disponibilizados. Nessa rede existem muitos tipos de computadores- PCs, Apple-Macintosh, UNIX e outros.

Servidores: são poderosos computadores permanentemente ligados à rede (Internet) que disponibilizam as informações, serviços, programas, etc.

Clientes: são os usuários que tem seus computadores conectados à Internet e fazem uso de informações que estão armazenadas nos servidores.

TCP/IP: é um protocolo de comunicação entre todos os computadores conectados à Internet. Enquanto o TCP (Transmission Control Protocol) é o responsável por dividir e enviar as informações através da rede, o IP (Internet Protocol) se responsabiliza por encontrar um caminho, através da rede, e fazer com que cada pacote de informação chegue a seu destino. Neste percurso as informações passam por equipamentos chamados roteadores cuja finalidade e fazer com que cada informação chegue ao endereço certo. 

Protocolo: Conjunto de regras que determinam a forma de operação entre os computadores para troca de dados.

World Wide Web: é uma enorme quantidade de páginas, denominadas páginas Web, que se distribuem pelos servidores na Internet. Uma página Web é essencialmente constituída por textos e imagens, mas pode ter sons, animações, filmes e programas interativos. Foi criada em 1991 num Laboratório de Física de Partículas, na Suíça, pelo físico Tim Berners Lee, porém no outono de 1969, dois computadores cada qual instalado em uma universidade americana foram interligados. Para acessar uma página Web necessitamos de um programa instalado em nosso computador denominado Browser (navegador). Os navegadores mais utilizados são: Netscape Navigator e o Microsoft Internet Explorer. No início a Web só era capaz de exibir textos e não apresentava uma interface amigável, ou seja, a navegação para a maioria dos usuários era muito complicada. Em 1993 foi desenvolvido o programa Mosaic (primeiro navegador gráfico), pelos estudantes da Universidade de Illinois, que acabava com a navegação textual e introduzia uma navegação mais amigável. Foi com a liberação para uso comercial que a Internet ganhou um impulso significativo.

Site: é um conjunto de páginas Web armazenados em um servidor. A página inicial através da qual se acessa o site é denominada HOME PAGE.   

Página Web: é constituída por textos e marcadores(comandos) especiais denominados "tags" de uma linguagem de programação,simples, chamada HTML (Hypertext Markup Language). Sua principal função é formatar o texto a ser exibido e criar as ligações entre as páginas da Web. O código fonte deve ser escrito observando-se as regras de sintaxe da linguagem, posteriormente ele será interpretado pelo navegador que por sua vez se encarregara de executar os marcadores (tags) do código, afim de, formatar os textos e acessar os recursos da Web. O programa Notpad (Bloco de Notas) do Windows é o mais simples dos editores utilizados para escrever o código fonte da linguagem HTML. Editores de HTML mais modernos, permitem a diminuição do tempo de geração de uma página, pois através de botões, permitem a inserção dos marcadores diretamente no documento que está sendo criado.

HTML: não é uma linguagem de programação, mas sim de marcação. Usando a codificação pré-definida (TAGS) conseguimos informar ao navegador como queremos que ele formate a aparência do documento. Cada TAG do documento deverá estar dentro dos sinais de menor que (<) e maior que (>). Exemplo <HTML>. Esses marcadores podem ser únicos (início) ou duplos (início e fim). Exemplo: único => <BODY>    duplo => <HEAD> ... </HEAD>
