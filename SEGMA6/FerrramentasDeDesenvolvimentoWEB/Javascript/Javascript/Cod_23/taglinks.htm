TAGS para inserção de Links

Uma das grandes vantagens da Internet é a confecção de documentos com a aplicação do conceito de Hipertexto. Através dele você liga o seu documento a outros por meio de vínculos especiais chamados HIPERLINKS. 

Inserindo links em um documento HTML
Para inserirmos um link em um documento HTML devemos utilizar a TAG < A HREF = "c:/hp/pagina1.html "> inicio <A>.

onde:
c:/hp/pagina1.html => corresponde ao local e o nome do documento com o qual desejamos nos conectar.
inicio => corresponde ao marcador para o qual o fluxo de navegação será desviado dentro do documento denominado pagina1.html que está no HD "C:\" na pasta "hp".

Exemplos de Links

a) Link de seu documento para outra home page da Internet
<a href="http://www.cade.com.br/"> O Cadê é um Site de Busca </a> => A frase " O Cadê é um Site de Busca" estará sublinhada e ao ser clicada irá fazer a conexão com o endereço http://www.cade.com.br. Não esqueça de colocar o  endereço completo.

b) Link de seu documento para outro documento
<a href="c:\hp\historico.htm"> Clique aqui para acessar o histórico do projeto </a> => Você está na página principal (home page) e quer que quando alguém clique na frase " Clique aqui para acessar o histórico do projeto" o fluxo seja desviado para a página historico.htm que está no HD de seu computador, ou seja, em c:\hp.

c) Link de seu documento para abrir o serviço de e-mail
<a href="mailto:ogiorgi@hotmail.com"> Clique aqui e me envie um e-mail </a> => Este link é para ser utilizado sempre que você desejar que o visitante lhe mande um e-mail. Após o atributo "mailto:" coloque o seu e-mail.

d) Link de seu documento para um trecho do mesmo, ou de outro documento
<a href="#topico1">  Tópico1 </a> => Cria um link para um indicador(âncora) chamado topico1 que está no interior do próprio documento.
<a name="topico1">Tópico1</a> => o atributo name coloca o indicador(âncora) no trecho do documento (topico1)  transformando-o e  um ponto de chegada de uma conexão hipertexto.

e) Link que permite, ao visitante, baixar (fazer o download) um arquivo compactado ou até mesmo um programa
<a href="apostila.zip">  Download Apostila </a> => Para que o visitante do seu site consiga fazer o download da apostila ele deve clicar no link " Download Apostila" desde que, você já tenha, previamente, feito o upload para o seu provedor do arquivo apostila.zip. 

Notas:
a) Utilize em seus links sempre textos curtos e que sejam significativos;
b) Na tag <body> você pode especificar as cores dos links, visitados, não visitados e ativos através dos atributos; Vlink, Link e Alink, respectivamente.
